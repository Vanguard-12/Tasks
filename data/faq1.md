# Lecture 1: Introduction to Python

In this lecture we covered the basics of Python programming, including variables, data types, and control flow statements such as `if`, `for`, and `while`. Remember that Python uses indentation to define code blocks.

**Key points**:
- Variables are created by assignment, e.g., `x = 10`.
- Common data types: `int`, `float`, `str`, `list`, `dict`.
- Control flow: `if/elif/else`, loops with `for` and `while`.
