# Lecture 2: Functions and Modules

This lecture introduced how to define functions using the `def` keyword, pass arguments, and return values. We also discussed how to organize code into modules and packages.

**Key points**:
- Define a function: `def my_func(arg1, arg2):`.
- Return values with `return`.
- Import modules with `import module_name` or `from module_name import func`.
- Packages are directories with an `__init__.py` file.

Ask if you need clarification on any of these concepts.
